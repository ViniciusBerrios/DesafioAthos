Como rodar o projeto Desafio Athos do Vinícius

##Instalação

	1 - Faça o clone deste projeto com git clone https://github.com/ViniciusBerrios/DesafioAthos.git

	2 - Entre no terminal do projeto no Visual Studio Code e instale as dependências com npm install

	3 - Rode a aplicação no terminal novamente com npm start


##Testando a aplicação

	1 - Abra o browser e visite a página http://localhost:3000/


##Navegação da aplicação
	
	1 - Navegue pelas páginas pelo Menu no canto superior direito
